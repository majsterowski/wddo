Js

Podstawowe zagadnienia

var nazwa_zmiennej


let nazwa_zmiennej

Deklaracja zmiennej.

Zmienne deklarowane typem var działają między klamrami funkcji, w jakiej zostały zadeklarowane, oraz każdej istrukcji zawartej w tej funkcji.

Zmienne deklarowane typem let działają między klamrami instrucki, w jakiej zostały zadeklarowane, nie wliczając nadrzędnych i podrzędnych instrukcji.

var nazwa_tablicy = new Array(nr)

var nazwa_tablicy = new Array(e1, e2, e3, ...)


var nazwa_tablicy = [e1, e2, e3, ...]


nazwa_tablicy[] = wartość

Deklaracja tablicy.

Między nawiasami podajemy ilość elementów, albo wartości przechowywane w tablicy.

Przypisując wartość do elementu tablicy lub wywołując go, w nawiasach kwadratowych, podajemy index danego elementu (pierwszy element posiada index 0).

document


document.write()

Obiekt pozwalający modyfikowanie dokumentu html.

document.write() wypisuje na stronie podaną treść. Wywołanie tej metody nie nadpisuje poprzedniej treści.

document.getElementById()


document.getElementById().innerHTML = wartość;

document.getElementById().innerHTML += wartość;

document.getElementById() znajduje element o podanym w nawiasie ID i go "chwyta".

document.getElementById().innerHTML zmienia kod html danego elementu na stronie.

Użycie samego znaku = nadpisze poprzednią wartość, aby dodać fragment kodu, należy użyć znaku += przed wpisaniem wartości.

Js

Okna dialogowe

alert()

Wyświetla komunikat na stronie o treści, podanej w nawiasach.

prompt ( " ", nazwa_zmiennej )


var x = prompt ( " " )

Wyświetla komunikat z polem tekstowym, następnie zapisuje wprowadzoną do niego wartość.

prompt() posiada dwa argumenty. Pierwszy to tekst wyświetlany w oknie, natomiast drugi to nazwa zmiennej, w której zostanie przechowana odpowiedź.

confirm( " " )

Wyświetla komunikat podany w nawiasach i zwraca wartość TRUE, albo FALSE.

Js

Zmienne tekstowe

var nazwa = " string ";


var nazwa = " Ala ma kota ";

Deklaracja zmiennej tekstowej (string).

nazwa.length

Zwraca długość tekstu przechowywanego w danej zmiennej.

Liczy też odstępy między słowami (spacje)!

nazwa.charAt ()

Zwraca znak, o podanym między nawiasami indeksie.

Pierwszy znak ma indeks 0!

nazwa.indexOf ()


nazwa.lastIndexOf ()

Zwraca indeks pierwszy lub ostatni (.lastIndexOf) podanego, między nawiasami, znaku.

nazwa.slice(index, n)

"Wycina" fragment tekstu ze zmiennej.

Pierwszy argument informuje, od którego indeksu zaczyna się fragment.

Drugi argument informuje, ile znaków wyciąć (a nie do którego indeksu!).

nazwa.toUpperCase()

Zmienia wielkość liter, w danej zmiennej, na wielkie.

nazwa.toLowerCase()

Zmienia wielkość liter, w danej zmiennej, na małe.

nazwa.fontcolor()


nazwa.fontcolor("green")

Zmienia kolor czcionki dla tekstu danej zmiennej, na podany między nawiasami.

nazwa.bold()

Pogrubia czcionkę dla tekstu danej zmiennej.

Pętle

for (var i=0; i<10; i++)

{

...

}


for ( ; ; )

{

...

}

Deklaracja pętli for.

Pierwszy argument pętli informuje od jakiej wartości rozpoczyna się pętla.

Dopóki drugie wyrażenie jest prawdziwe, pętla będzie się wykonywać.

Trzeci argument pętli informuje o modyfikacji zmiennej w każdym cyklu pętli.

W przypadku wpisania dwóch średników, między nawiasami, tworzy się nieskończona pętla.

while ( )

{

...

}

Deklaracja pętli while.

Pętla będzie się wykonywać dopóki wyrażenie między nawiasami będzie prawdziwe.

do

{

...

}

while ( )

Deklaracja pętli do_while.

Od pętli while, różni się tylko tym, że kod pomiędzy klamrami wykona się przynajmniej raz.
