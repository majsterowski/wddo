Selektory

*

{

margin: 0;
padding: 0;

}

Tak jak podano na przykładzie po lewej * jest selektorem wszystkich elementów na stronie.

Na samym początku pisania strony warto wyzerować marginesy wszystkim obiektom na stronie.

div

{

...

}

W tym przypadku div jest selektorem znacznika html a wszystko pomiędzy klamrami deklaracją.

#menu

{

...

}

W tym przypadku #menu jest selektorem ID, ponieważ nazwę znacznika poprzedza znak #.

.button

{

...

}

.button jest selektorem klasy, ponieważ nazwa jest poprzedzona znakiem .

div, #menu, .button

{

...

}

Teraz div, #menu i .button będą przyjmować takie same właściwości.

div p

{

...

}

Wszystkie p znajdujące się w jakimkolwiek znaczniku div przjmą wartości podane w klamrach.

Tj. wszyskie znaczniki, nawet pośrednio, np. <div><p></p><span><p></p></span></div>.

div > p

{

...

}

Wszystkie p bezpośrednio znajdujące się w jakimkolwiek znaczniku div przjmą wartości podane w klamrach.

Tj. wszyskie znaczniki <div><p></p><span></span></div>.

div [atrybut = "wartość"]

{

...

}

div > p:nth-child(nr)

{

...

}

Podane wartości przyjmie jedynie pierwszy/itd.(w zależności od nr w nawiasie) znacznik p, znajdujący się bezpośrednio w znaczniku div.

CSS

Kolory

red

cyan

aqua

Kolor opisywany słownie.

# ff0022

# f02

# 000000

# 000

Kolor opisywany szesnastkowo. Pierwsze dwie cyfry po znaku # to kolor czerwony, kolejne dwie to zielony i ostatnie to niebieski.

Jeśli obie cyfry każdego koloru się powtarzają można zapisać po jednej z nich;

rgb(255, 0, 0)

rgba(255, 0, 0, 1)

rgba(255, 0, 0, 0.5)

Pierwsza wartość to czerwony, druga to zielony a trzecia to niebieski. Każdy kolor przyjmuje wartości od 0 do 255.

rgba() posiada czwartą wartość, którą jest przeźroczystość. 0 oznacza całkowicie przeźroczysty a 1 w pełni widoczny.

1 mm

1 pt

1 pc

1 in = 2.54 cm

1 in = 96 px

1 in = 72 pt

1 pc = 12 pt

1 %

1 vw

1 vh

1 vmin

1 vmax

1 em

1 ex

Jednostki responsywne.

Jeden vh (viewport height) to 1% wysokości przeznaczonej na wyświetlenie strony.

Jeden em to rozmiar aktualnej czcionki.

CSS

Background

background-color: red;

background-color: #f00;

background-color: rgb(255, 0, 0);

Zmienia kolor tła na podany color;

background-image: url("bg.png");

background-position: left top;

background-repeat: no-repeat;

background-repeat: repeat-x;

background-repeat: repeat-y;

background-image ustawia podany obraz jako tło strony.

background-position ustawia pozycję tła.

background-repeat ustawia czy i jak ma powtarzać się obraz.

background: linear-gradient(45deg, silver, black);

background: radial-gradient(circle, silver, black);

Ustawia gradient jako tło strony.

opacity: 1;

Ustawia przeźroczystość.

0 oznacza całkowitą przeźroczystość, 1 w pełni widoczny obiekt.

CSS

Obiekty blokowe

display: block;

display: inline-block;

display: flex;

Zmienia fizyczne zachowanie elementu na stronie.

width: 10px;

width: 10vw;

width: 10%;


max-width: 10vw;

min-width: 10vw;

Zmienia szerokość obiektu. Nie trzeba podawać statycznej szerokości, możemy określić minimalny oraz maksymalny zakres.

height: 10px;

height: 10vw;

height: 10%;


max-height: 10vw;

min-height: 10vw;

Zmienia wysokość obiektu. Nie trzeba podawać statycznej wysokości, możemy określić minimalny oraz maksymalny zakres.

position: static;

position: absolute;

position: relative;

position: fixed;

Pozycja statyczna jest domyślną pozycją, w której obiekt ustawia się w odniesieniu do innych elementów blokowych.

Pozycja relatywna sprawia, że obietk utawia się w odniesieniu do swojej pozycji.

Pozycja absolutna ustawia obietk w odniesieniu do nadrzędnego obiektu relatywnego lub jeśli takiego nie ma, do maksymalnego lewego, górnego punktu strony.

Pozycja fixed "przykleja" obiekt do danego miejsca na ekranie i nie zmienia go, pomimo scrollowania strony.

top: 20px;

right: 20px;

bottom: 20px;

left: 20px;

Ustawiają odległość od danej krawędzi obiektu nadrzędnego.

Opcje te nie działają ze statyczną pozycją!

margin-left: 10px;

margin-top: 10px;

margin-right: 10px;

margin-bottom: 10px;


margin: 10px;

margin: 10px 10px;

margin: 10px 10px 10px;

margin: 10px 10px 10px 10px;

Są to marginesy zewnętrzne. Ustawiają odległość od danej krawędzi obiektu nadrzędnego.

Przy wpisaniu tylko jednej wartości we właściwości margin, przyjmują ją wszystkie cztery marginesy;

Przy wpisaniu dwóch wartości we właściwości margin, pierwsza oznacza marginesy wertykalne a druga marginesy horyzontalne;

Przy wpisaniu trzech wartości we właściwości margin, pierwsza oznacza margines górny, druga marginesy poziome, a trzecia margines dolny;

Przy wpisaniu czterech wartości we właściwości margin, pierwsza oznacza margines górny, druga margines prawy, trzecia margines dolny, a czwarta margines lewy;

padding-left: 10px;

padding-top: 10px;


padding: 10px;

padding: 10px 10px 10px 10px;

Padding to margines wewnętrzny. Określa odległość elementów w środku obiektu od jego krawędzi.

Kolejność podawania wartości jest taka sama jak we właściwości margin (z ruchem wskazówek zegara).

border: 2px solid #fff;

border: 2px dashed #fff;

border: 2px dotted #fff;

border: 2px double #fff;


border: 2px groove #fff;

border: 2px ridge #fff;

border: 2px inset #fff;

border: 2px outset #fff;

Obramowanie elementu blokowego.

Tak jak w margin oraz padding możemy wyodrębnić części obramowania (border-top, itp.).

Przy skróconym zapisie (na przykładach) nie można pominąć stylu obramowania (solid, double, groove, itp.)

Pierwsze cztery style przedstawiają obramowanie "2D" a kolejne "3D".

box-shadow: 5px 5px 5px gray;

Cień obiektu.

Pierwsza wartość oznacza przesunięcie w poziomie, druga w pionie, trzecia rozmycie, a czwarta kolor.

box-sizing: border-box;

box-sizing: content-box;

border-box wlicza marginesy, paddingi, itp. w wymiary elementu blokowego.

content-box natomiast dolicza marginesy, paddingi, itp. do wymiarów elementu blokowego.

float: left;

float: right;

Właściwość ta sprawia, że obiekt przylega w poziomie do podanej krawędzi elementu nadrzędnego lub sąsiedniego.

clear: left;

clear: right;

clear: both;

Zamyka obszar działania właściwości float.

Po zakończeniu ustawiania obiektów warto dodać na końcu element np. z clear: both;

font-style: italic;

font-variant: small-caps;

Zmienia czcionkę na kapitalik.

text-align: left;

text-align: right;

text-align: center;

text-align: justify;

Justuje tekst.

text-decoration: line-through;

text-decoration: overline;

text-decoration: underline;

text-decoration: none;

text-indent: 10px;

Ustawia wcięcie pierwszej lini akapitu;

text-shadow: 2px 2px 2px gray;

Ustawia cień dla tekstu. Działa na tej samej zasadzie co box-shadow.

text-transform: capitalize;

text-transform: uppercase;

text-transform: lowercase;

color: #fff;
